## mediaId Type

`string` ([mediaId](character-properties-media-media-properties-mediaid.md))

## mediaId Constraints

**pattern**: the string must match the following regular expression:&#x20;

```regexp
^([-_a-z0-9]+)(:[-_a-z0-9]+)?$
```
